Overview

Resource representations

JSON

{
  "kind": "chromewebstore#item",
  "id": string,
  "publicKey": string,
  "uploadState": string,
  "itemError": [
    (value)
  ]
}
Property name Value Description Notes
id string the unique id of the item.
itemError[] list The detail human-readable status of the operation.
kind string This is an item in the chrome webstore.
publicKey string the public key of this item.
uploadState string The status of the operation.

Acceptable values are:
  • "FAILURE":
  • "IN_PROGRESS":
  • "NOT_FOUND":
  • "SUCCESS":

patch
Updates an existing item. This method supports patch semantics.
publish
Publishes an item.
update
Updates an existing item.
