Web store API Reference

This API reference is organized by resource type. Each resource type has one or more data representations and one or more methods.

Resource types

  1. InAppProducts
  2. Items
  3. Licenses
  4. Payments

InAppProducts

For InAppProducts Resource details, see the resource representation page.

Method HTTP request Description
URIs relative to https://www.googleapis.com/chromewebstore/v1.1, unless otherwise noted
get GET  /items/itemId/skus/sku Gets the in-app product information of an item.
list GET  /items/itemId/skus Lists the in-app product information of an item.

Items

For Items Resource details, see the resource representation page.

Method HTTP request Description
URIs relative to https://www.googleapis.com/chromewebstore/v1.1, unless otherwise noted
get GET  /items/itemId Gets your own Chrome Web Store item.
insert POST
https://www.googleapis.com/upload/chromewebstore/v1.1/items 		Inserts a new item.
patch PATCH  /items/itemId Updates an existing item. This method supports patch semantics.
publish POST  /items/itemId/publish Publishes an item.
update PUT
https://www.googleapis.com/upload/chromewebstore/v1.1/items/itemId
and
PUT  /items/itemId 		Updates an existing item.
verify POST  /items/verify Verify whether items are hosted on Chrome Web Store.

Licenses

For Licenses Resource details, see the resource representation page.

Method HTTP request Description
URIs relative to https://www.googleapis.com/chromewebstore/v1.1, unless otherwise noted
get GET  /licenses/appId/userId Gets the licenses for Chrome hosted apps.
getUserLicense GET  /userlicenses/appId Gets the licenses for Chrome packaged apps.

Payments

For Payments Resource details, see the resource representation page.

Method HTTP request Description
URIs relative to https://www.googleapis.com/chromewebstore/v1.1, unless otherwise noted
buy POST  /payments/buy Inserts a cart and returns the JWT.
list GET  /items/itemId/payments Lists the in-app products that the user has purchased.
